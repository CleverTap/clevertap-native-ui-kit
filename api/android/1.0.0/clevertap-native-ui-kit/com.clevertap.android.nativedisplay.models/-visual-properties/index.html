VisualProperties

@Immutable
data class VisualProperties(val background: Background?, val backgroundColor: String?, val opacity: Float?)

Visual styling properties for containers and visual elements. Used internally by renderer for background and opacity.

Contains non-cascading visual properties:

  • background: Complex background (gradients, images, animations)

  • backgroundColor: Simple solid background color

  • opacity: Element opacity (0.0 to 1.0)
