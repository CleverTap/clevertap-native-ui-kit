bind

fun bind(cleverTapApi: CleverTapAPI, forwardTo: DisplayUnitListener? = null): Boolean

Bind the bridge to a CleverTap API instance.

Registers a composite display unit listener that forwards to both the bridge and an optional client listener. This avoids replacing the client's existing listener, since the Core SDK only supports a single DisplayUnitListener.

// Without client listener
bridge.bind(CleverTapAPI.getDefaultInstance(context)!!)

// With client listener (both receive callbacks)
bridge.bind(CleverTapAPI.getDefaultInstance(context)!!, forwardTo = myListener)

Return

true if binding succeeded, false otherwise

Parameters

cleverTapApi

The CleverTapAPI instance to wire to.

forwardTo

Optional client DisplayUnitListener to forward raw display units to. If provided, it receives the same callback the Core SDK would normally deliver, preserving the client's existing display unit handling.
